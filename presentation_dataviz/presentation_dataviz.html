Why Python and Javascript are a perfect match for dataviz?


  • Browsers are a natural home for advanced visualizations
  • Javascript is and, for foreseeable future will be, #1 language for interactive dataviz
  • Python is very effective in data wrangling
  • Being an multi-purpose language, Python supports overall toolchain better than e.g. R

Data analysis toolchain

Dataviz in browser

The idea is to harness browser for presenting data. This is done by binding data to the Document Object Model (DOM) that describes the browser window.

The best way to do this is Javascript library d3.



One does not have to become a full-fledged web developer, but some HTML and CSS is a must.
