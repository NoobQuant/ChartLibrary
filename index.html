Chart Library

Select chart from the dropdown menu.

  1. Financial Cycles
    Financial cycle plots from the paper Wavelet decomposition of the financial cycle: An early warning system for financial tsunamis - Bank of Finland Discussion Paper Series 11/2017.

  2. Applied Data Science
    Course work visualizations from the course Applied Data Science in Python taken at University of Mischigan at Coursera.


See respective chart for more information. The chart library framework is based on the startmin dashboard template. Most charts are produced using the d3 Javascript library with the help of the excellent book D3 Tips and Tricks by d3noob as well as many examples found online. Particular source links are given in the source code. The Chart Library unfortunately doesn't yet fully support mobile devices.
